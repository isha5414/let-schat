# Chat App

Fullstack Chat Application using Node.js, Express, MongoDB, and React (Vite).

## Features
- User authentication (register/login)
- Real-time messaging
- Chat rooms
- Responsive frontend

## Backend
- Express server
- MongoDB database connection
- JWT authentication

## Frontend
- React with Vite
- Components for chat UI

## Run Locally

### Backend
```bash
cd backend
npm install
npm start
